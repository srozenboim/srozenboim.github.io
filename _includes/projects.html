Swill - Front End, Back End

A mobile application that allows users to create cocktails without using any measuring tools. A user can view a list of drinks which are categorized by the type of liqueur inside of that cocktail. A user can then see a recipe for that drink, and a "pour guide". The pour guide shows the user a visual with horizontal lines across the screen. The user can then place their phone directly next to their glass, and pour each ingredient in the drink until it reaches the line. Then do the same with every other ingredient... Technologies used: React Native, NodeJS, ES6, Ruby on Rails, PostgresQL.
